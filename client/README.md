# MeetHub App - Frontend Client

This is the React frontend for the MeetHub App, built on top of Vite.

## Features Included
- Login & Registration UI (auth state stored in localStorage)
- Group Video conferencing room layout
- Stable peer-to-peer connection initialization via WebRTC
- Text Chat widget integration
- Custom collaborative Whiteboard modal (draw, erase, clear, and download canvas)
- File sharing widget with direct backend file upload support

## Tech Stack
- React
- Socket.io-client
- Axios
- HTML5 Canvas & MediaDevices API

## Commands
- `npm install`: Install dependencies.
- `npm run dev`: Start the Vite dev server locally.
- `npm run lint`: Run ESLint analysis.
- `npm run build`: Compile the build artifacts into the `dist` directory.
